DOJ FOIA Records

Background

Data comes from this FOIA request on muckrock.com, which asks for the "FOIA request log covering 2017, which includes a field showing the subject of each request and a field for the final disposition."

How long did requests take to process?

Each line represents the duration of a FOIA request. It exists from the day it was submitted until it was completed and is color coded based on the "disposition" (outcome) of the request.
